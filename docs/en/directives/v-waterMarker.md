# v-waterMarker

## Function Introduction

Add custom watermark

## Example

<div class="watermark" v-waterMarker="{text:'Eonova copyright',textColor:'rgba(180, 180, 180, 0.4)'}"></div>

## Usage

```typescript {3-6}
<template>
  <div
    v-waterMarker="{
      text:'Eonova copyright',
      textColor:'rgba(180, 180, 180, 0.4)'
    }"
  ></div>
</template>
```

<style scoped>

  width: 100%;

}
